# PrintInvoice
Expand the invoice API to access the printable PDF (base64 encoded) via the Magento API

(GET)  /pub/rest/default/V1/invoices/{id}/print

| **Parameter** | **Required** | **Type** | **Description** |
|:-------------:|:------------:|:--------:|:---------------:|
| id            | required     | integer  | The invoice ID  |
